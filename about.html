MyLibraryTracker

This app is based on the Library project assignment from The Odin Project. If you love to read and want to keep track of what you have already read or want to read, use this project for free!

Features/Languages Used

  • Vanilla Javascript, HTML, CSS
  • Book carousel with page flip transitions
  • Local storage used for personal library list
  • Exporting/importing CSV book data
  • Search hundreds of books by title, author, or keywords using the https://openlibrary.org api
  • Add new books directly from search results, or add books manually by form
  • Keep track of which books you have read with the click of a button
  • Responsive and mobile friendly
